## Features

- Cache storage
- Wallet pokedex
- Cool design
- Search pokemon by name


## Installation

Before installing the front end, you must already have the backend running. See more here: https://github.com/yuripinheirot/pokemon-project-back

Installation

```bash
  cd pokemonProject
  npm install  
  npm start
```

Just it!
